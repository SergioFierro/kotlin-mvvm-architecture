## About
This project allow you to search for products using MercadoLibre API, you will find a MVVM architecture with kotlin, hilt, retrofit, coroutines

## Requirements
* Android Studio 4.0 or higher
* Java 8
* Android 6.0 (23) SDK or higher
* Gradle 6.1.1
* Kotlin 1.3.72
